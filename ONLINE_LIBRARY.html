ONLINE LIBRARY

Project advance programming to create an online library.

abstract

Development of a online library system, integrating a Flask-based API backend with a PyQt desktop application frontend. The project leverages AI-powered tools, specifically Copilot and ChatGPT, for accelerating the generation of core backend logic, Qt Designer-based GUI components, and the Python integration code facilitating communication between the frontend and backend via HTTP requests.

The implemented system provides comprehensive functionalities for media management, including dynamic search and sorting capabilities across book attributes (name, author, publication year, category), as well as mechanisms for adding new media items, and managing their availability through rental and return operations. All library data is structured as a list of Python dictionaries within an in-memory database. To ensure the reliability and functional integrity of the system, a multi-tiered testing strategy was adopted, encompassing dedicated unit tests for the backend API, initialization validation for the GUI frontend, and end-to-end integration tests confirming seamless data flow and interaction between client and server components.
